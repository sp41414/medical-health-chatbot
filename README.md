# Medical Health Chatbot

## How to setup

1. Clone the repo:

```bash
git clone https://github.com/sp41414/medical-health-chatbot.git
```

2. Add your API key in [script.js](./script.js)

3. Run in the project directory

```bash
npm install
```

4. Host on localhost

```bash
python3 -m http.server
```
